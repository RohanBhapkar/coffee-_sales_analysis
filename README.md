 ☕ Coffee Shop Sales Analysis


 📌 Project Overview

This project focuses on cleaning, processing, and visualizing a Coffee Shop Sales dataset using Python. The objective is to perform data preprocessing, identify business trends, and generate meaningful insights through data visualization.


 🎯 Objectives

* Handle missing values
* Remove duplicate records
* Detect and remove outliers
* Perform feature engineering
* Create visualizations using Matplotlib and Seaborn
* Generate business insights from sales data


 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook


 📂 Dataset Information

The dataset contains coffee shop transaction records, including:

* Transaction ID
* Transaction Date
* Transaction Time
* Product Category
* Product Type
* Product Detail
* Store Location
* Unit Price
* Transaction Quantity


 🧹 Data Cleaning Process

* Checked for missing values
* Removed duplicate records
* Converted date columns into proper format
* Created new features:

  * Revenue
  * Month
  * Day
  * Hour
* Removed outliers using the IQR (Interquartile Range) method


 📊 Visualizations Created

* Revenue by Store Location
* Top 10 Products by Revenue
* Revenue by Product Category
* Monthly Revenue Trend
* Hourly Sales Trend
* Revenue Share Pie Chart
* Correlation Heatmap
* Revenue by Day of Week


 📈 Key Insights

* Identified the highest revenue-generating store location.
* Determined the best-selling product category.
* Found the top-performing products.
* Analyzed peak sales hours.
* Observed monthly and daily sales trends.
